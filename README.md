PC Builder Studio | AI-Powered PC Compatibility & Pricing System
Developed a web application that allows users to build custom PCs while automatically checking
CPU–Motherboard–RAM–GPU compatibility in real time.
Built an AI chat assistant (RAG-based) to guide users with recommendations, performance
insights, and compatibility queries.
Implemented component catalog with categories, price calculations, and build comparison
features.
Created user dashboard to save, edit, and manage multiple PC builds.
Integrated JWT authentication and OAuth login options for secure access.
Integrated Razorpay payment gateway with test/live modes and order verification workflows.
Designed and developed backend RESTAPIs and database models for users, builds, components,
chat logs, and payments.
Followed MVC architecture, modular coding practices, and implemented validation and error
handling across modules.
Tech Stack: React.js, Node.js, Express.js, MongoDB, Mongoose, RESTAPIs, Axios, JWT, OAuth, Razorpay,
Tailwind/MUI, AI RAG
Key Outcomes: Reduced user errors in PC building, improved decision making, and delivered a scalable
platform for hardware configuration
